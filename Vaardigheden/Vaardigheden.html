HTML-CSS

HTML staat voor Hyper Text Markup Language, en wordt gebruikt om de structuur van een website te bepalen. HTML is een opmaak taal, dat betekent dat de DOM de code van boven naar onder leest. Samen met HTML werkt CSS, CSS wordt gebruikt om de opmaak van de website te bepalen. Zonder CSS zou je alleen maar de structuur zien.

Javascript

Javascript wordt gebruikt om de website interactief te maken. Integendeel tot HTML is javascript een scripttaal. De meeste webapplicatie maken gebruik van Javascript.

PHP

PHP volledig uitgeschreven Hypertext Preprocessor is net zoals Javascript een scripttaal. En wordt onder andere gebruikt om informatie te verwerken tot hypertext. Dat kan ook heel makkelijk met PHP dat komt omdat PHP een server-side scripting taal is. In tegendeel tot javascript en HTML, loopt PHP op de server zelf.
